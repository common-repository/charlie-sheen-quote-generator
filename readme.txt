== Description ==

This plugin displays a random quote from Charlie Sheen. This includes a Widget you can include in your sidebar, a short code, [charlie sheen] that you can use in your blog posts, and also replaces the default Hello Dolly quote in the Wordpress admin panel with a quote from everyone's favorite actor. Winning!

http://www.leonamarant.com/2011/03/03/charlie-sheen-quotes-wordpress-plugin/


== Installation ==

1. Upload the charlie-sheen folder to the /wp-content/plugins/ directory
2. Activate the plugin through the 'Plugins' menu in WordPress

Once Uploaded and Activated you can:

- Use the Charlie Sheen Quote Generator Widget in your sidebars
- Use the short code <strong>[charlie sheen]</strong> in your posts for a random quote.
- Enjoy Vintage Balderdash.
